Play with some Object Stuff in the Console!

    Object Enhancement Practice

  • createInstructor(firstName, lastName) accepts two strings and should return an object with the firstName key set to your input, and the lastName key set to your input.
  • If you look at instructor[42] you'll see "That is my favorite!".
  • newInstructor.sayHi() should return "hi!" in the console.
  • newInstructor.sayBye() should return "Colt says bye!" in the console.
  • createAnimal(species, verb, noise) accepts three strings and should return an object with the species set to your input, and the verb set to a function that returns noise in the console.
